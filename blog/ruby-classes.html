An Example Class

September 24, 2015

Classes are an essential part of the programming heirarchy. Defining a class allows you to group behaviors (methods) together so that you can quickly create objects that behave the same way. Classes act as a convenient way to bundle certain attributes and actions together into a single object. Everything done in Ruby is essentially a an object, and classes are simply one way of creating an instance of objects.

Classes effectively organize methods and objects in a meaningful way. Classes exist for the purpose of being instantiated, or to create objects that are instances of the class. In doing this, you can create instance methods within the class, and instance variables that reside within the class. In the example below you can see how a bike has a given wheel size, frame, recommended tire pressure, and an owner. These different attributes of the bike are part of the intantiated Bike class.
